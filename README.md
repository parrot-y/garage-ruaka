# Garage Ruaka - Computerized Car Repair Services

Professional automotive repair website for Garage Ruaka, located in Kiambu County, Kenya.

## Features

- Responsive design optimized for mobile-first experience
- Modern UI with professional aesthetics
- Service showcase and portfolio
- Contact information and location details
- Blog section for automotive tips

## Technology Stack

- HTML5
- CSS3 (with animations and transitions)
- JavaScript
- Bootstrap framework
- Slick Slider for carousels
- Font Awesome icons

## Local Development

1. Clone this repository
2. Open `index.html` in your browser or use a local server
3. For live server: `python3 -m http.server 8080`
